# Computation of Heun functions via the Birkandan-Giscard-Tamar path-sum algorithm
An implementation of an algorithm to compute Heun functions, as presented in [Computations of general Heun functions from their integral series representations](https://ieeexplore.ieee.org/document/9598600) by T. Birkandan, P.-L. Giscard and A. Tamar.
